# Dphishbox

Desktop application to match URLs against phishing black-lists.

## Getting Started

* Download or clone
* create virtual env `virtualenv env` (optional)  
  activate virtualenv `\env\Scripts\activate`
* `cd dphishbox`
* `pip install -r requirements.txt`
* `python run.py`

## Running the tests

`pip install pytest`  
`pytest -v`
